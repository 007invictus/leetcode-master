
## 题目地址 

https://leetcode-cn.com/problems/valid-anagram/

## 思路

使用哈希法来判断 s 中的字母是否 都在t中，且t中的字符也都在s中 

## 代码
```
class Solution {
public:
    bool isAnagram(string s, string t) {
        int record[26] = {0};
        for (int i = 0; i < s.size(); i++) {
            record[s[i]-'a']++;
        }
        for (int i = 0; i < t.size(); i++) {
            record[t[i]-'a']--;
        }
        for (int i = 0; i < 26; i++) {
            if (record[i] != 0) {
                return false;
            }
        }
        return true;
    }
};
```
