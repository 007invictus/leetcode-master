## 题目地址 
https://leetcode-cn.com/problems/reverse-string/

## 思路

遍历数组的前一半，同时和后一半做交换就可以了

## 代码

```
class Solution {
public:
    void reverseString(vector<char>& s) {
        for (int i = 0, j = s.size() - 1; i < s.size()/2; i++, j--) {
            swap(s[i],s[j]);
        }
    }
};
```
